Тестовое приложение для магазина.

Бэкенд:
Язык: C#
Фреймворк: ASP.NET Core 9.0
База данных: SQL Server (Entity Framework Core)

Фронтенд
Язык: JavaScript (React)

Библиотеки:
React Router
Axios
Material-UI
Context API

Инструкция по запуску. Перейдите в BagStore.Backend и запустите сервер, так как приложение пока тестировочное, можете запустить сервер через Start.bat или командой dotnet run
Запустите приложение: Команда npm start
